# Cara Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/fikkyh/task-hidden-items.git
```

Masuk ke folder project

```bash
cd task-hidden-items
```

## 2. Jalankan Script

```bash
php hidden-item.php
```

## 3. Masukkan Input

Saat diminta, masukkan rentang langkah untuk tiap arah (contoh: `2` atau `2-5`, kosongkan untuk default `1-4`):

```
Masukkan jumlah langkah Utara/U
Masukkan jumlah langkah Timur/T
Masukkan jumlah langkah Selatan/S
```

## 4. Lihat Hasil

Script akan menampilkan daftar koordinat kemungkinan lokasi item beserta grid yang ditandai dengan `$`.
